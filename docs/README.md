# Documentation

This folder contains the main project documentation.

Contents:
- **hardware/**: hardware design, components, wiring, and assembly notes
- **software/**: software architecture, modules, and execution flow
- **firmware/**: microcontroller code structure, pin mapping, and control logic
- **testing/**: test procedures, results, issues, and improvements

Use these documents to understand how the robot is built, how it works, and how it was tested.
